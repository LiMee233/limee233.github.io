Windows 语言变成「中文（新加坡）」

2021/05/16
阅读时长: 2 分钟

问题概述

西元 2021 年 年初， Li Mee 在安装完 Windows 10 并登陆微软账户后，经常发现系统的语言自动会变成「中文（新加坡）」，这导致系统默认的时间格式和「中文（中华人民共和国）」的时间格式不同，使用起来实在是很难受。

关键是，系统设置并不能直接从「中文（新加坡）」改成「中文（中华人民共和国）」，毕竟都是中文嘛，所以……

临时解决方案

信息
灵感来自于：[https://www.sysgeek.cn/windows-10-force-remove-language-pack](https://www.sysgeek.cn/windows-10-force-remove-language-pack)

强制删除「中文（新加坡）」

使用管理员模式打开 Windows Powershell，输入命令： Get-WinUserLanguageList ，观察所得信息中是否包含 LanguageTag : zh-Hans-SG

若存在如上内容，接下来仅需要依次（一次一行）输入并执行以下命令，即可删除「中文（新加坡）」：

  1. $LangList = Get-WinUserLanguageList
  2. $MarkedLang = $LangList | where LanguageTag -eq "zh-Hans-SG"
  3. $LangList.Remove($MarkedLang)
  4. Set-WinUserLanguageList $LangList -Force

添加「中文（中华人民共和国）」

打开设置——时间和语言——语言——添加语言——搜索Chinese，选择「中文（中华人民共和国）」即可。

完美解决方案

如果多次重装系统，就会发现这样的问题会屡次发生。因此可以猜测此问题可能是「微软账户关于 Windows 设置同步的信息出现异常」导致的。

实际上也确实如此， Windows 10 系统的「同步你的设置」功能（位于“设置——账户——同步你的设置”）是造成此问题的元凶，不稳定的同步导致 OneDrive 上存放的用户设置出现了差错。

想要解决这个问题，就需要想办法从 OneDrive 上删除错误的用户设置同步信息。

删除用户设置

打开 删除用户设置的网站 （请在新标签页内打开，此外还需要魔法），按步骤删除同步的用户设置即可。

注意
删除同步的用户数据，并不会让系统内的语言“改邪归正”，还需要配合「临时解决方案」或选择重装系统。

其他

用户设置同步出现的差错还可能导致其他问题，目前还已知的有：「登录微软账户后，鼠标左右键被对调」

截至目前，如果发现登陆微软账户后设置发生了无法预知的错误更改，基本可以怀疑是「用户设置同步出错」导致的。
